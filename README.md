# :satellite: Cept! Auto Accept

Cept! is a lightweight Windows tray application that auto-accepts League of Legends matches and surfaces a simple status menu.

## Developing Cept!

The application lives in the [Conduit](./conduit) directory. Open `Conduit.sln` in Visual Studio to restore dependencies and build the app.
